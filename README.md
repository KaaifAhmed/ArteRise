# ArteRise: A Tech Company
ArteRise is a tech company that provides different IT solutions to businesses, this is the complete website that I created for that brand. This website is focused on getting customers and sales for the brand, while making sure the UI is interactive and modern, representing an IT company.

## Feature:
- Beautiful and elegant design
- Landing page for converting the customers
- Services page for highlighting all of the provided services
- Simple contact page for CTA

*This website served as an advancement in my design skills and taught me about modern designing, cool effect and converting copywriting.*
